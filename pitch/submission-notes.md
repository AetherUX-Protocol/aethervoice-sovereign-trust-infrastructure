# AetherVoice Submission Notes

## Overview

AetherVoice is a sovereign trust identity infrastructure
for high-risk commerce.

The project combines:

- SNS identity infrastructure
- QVAC local AI verification
- Umbra confidential settlement

to create persistent transactional trust identity.

---

## Core Concept

Traditional identity systems verify:
- ownership
- wallets
- profiles

AetherVoice verifies:
- transactional credibility
- settlement reputation
- behavioral trust signals

before value moves.

---

## Key Features

- AI-powered transaction verification
- Persistent SNS trust identity
- Local-first QVAC inference
- Confidential settlement via Umbra
- Reputation-aware commerce workflows
